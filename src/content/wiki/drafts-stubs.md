Not everything needs to be polished. This page describes a simple approach to drafts that keeps the wiki usable.

## Draft Notes

If youâ€™re not ready to write a full page:

## Stubs

For a true stub, use this structure:

```md
Summary sentence.

## TODO
- Fill in definition
- Add examples
- Link related topics
```
